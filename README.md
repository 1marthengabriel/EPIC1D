# EPIC1D - 1D Electrostatic Particle-in-Cell Code

Computational Plasma Physics lab code by Dr. David Dickinson, modified by Marthen Gabriel.

## Setup

Clone the repository:
```bash
git clone https://github.com/1marthengabriel/EPIC1D.git
cd EPIC1D
```

Create and activate virtual environment:
```bash
python3 -m venv venv_complab
source venv_complab/bin/activate
```

Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Code

**Lab Session 1-2:**
```bash
python3 src/epic1d.py
```

**Lab Session 2-3:**
```bash
python3 src/scan_epic1d.py
```
